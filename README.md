# ☕ Cafe Finder

Cafe Finder is a location-based web application that helps users discover nearby cafes and restaurants within a specified search radius. It provides an interactive map interface and dynamically displays nearby places based on user input.

## 🚀 Features

* Search cafes and restaurants by location
* Filter places within a specified radius
* Interactive map visualization using Leaflet.js
* Clickable location markers
* Dynamic search results panel
* Responsive and user-friendly interface
* Real-time place discovery using OpenStreetMap data

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript 
* Leaflet.js
* OpenStreetMap
* Overpass API

## ⚙️ How It Works

1. Enter a location.
2. Select a category (Cafe, Restaurant, or All).
3. Specify a search radius.
4. Click **Find Places**.
5. Nearby locations are displayed on an interactive map and in a results panel.
6. Selecting a result focuses the map on the chosen location.
